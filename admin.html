Vendedoras

Cada vendedora entra a su propia vitrina con código y clave. El precio de venta es el mismo para todas (se configura en Productos) — acá solo elegís qué categorías vende y su % de comisión. Sus ventas se envían solas a Odoo; acá puedes ver o forzar el cierre de la venta que tiene abierta.

Nueva vendedora

Las ventas siempre se crean en Odoo desde la cuenta admin — este email es solo para tus registros, no se manda a la API.

La comisión es lo que se lleva la vendedora sobre la ganancia de cada venta (precio de venta − costo del proveedor) × este %. El precio de venta lo fija Productos, es igual para todas.

Productos

Fase 2: Sistema de costos y márgenes. Por cada producto puedes definir Costo, Precio PVP (precio de venta al público), IVA%, y Comisión override (si quieres que esta categoria/producto tenga diferente % que el de la vendedora). El margen bruto se calcula automático. Lo que apagues en "Disponible" desaparece de todas las vitrinas.

Categorías

Multiplicador: precio de venta por defecto = costo del proveedor × este número, para lo que no tiene precio fijo (sin multiplicador propio, una categoría usa ×2). Ícono y nombre son opcionales — "Auto" adivina el ícono por el nombre de la categoría; si no queda lindo, elegí uno de la lista o escribí un nombre más corto para mostrar.

Proveedores

Cada proveedor conecta el portal a su propio catálogo — hoy vía Odoo (URL, base de datos, usuario y clave), o "Manual" para uno que todavía no tiene Odoo. Agregar un proveedor acá no cambia nada solo — hasta que se conecte el catálogo, no aparece en ninguna vitrina.

Información adicional de producto

Odoo (o el catálogo manual) aporta el código, precio, stock y foto. Aquí subes un Excel con la información que no tiene (características, garantía, cómo venderlo…) y aparece en la ficha de cada producto, igual para todas las vendedoras.

Cómo armar tu Excel

La primera columna debe ser el Código (el mismo SKU de Odoo). Si para alguna fila no tienes el código a mano, puedes dejarlo vacío y usar la columna Código de barra en su lugar — se busca solo en el catálogo. Las demás columnas son libres: tú les pones el título que quieras, y cada una aparece como un dato en la ficha. Puedes subir varios Excel distintos (uno para relojes, otro para calculadoras, etc.).
